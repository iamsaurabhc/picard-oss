# Phase 3 evaluation

## Automated

Use the project **venv** (`backend/.venv`), not system `python`/`pip`:

```bash
cd backend
source .venv/bin/activate
python -m pytest tests/test_citations.py tests/test_chat_stream.py -q
python scripts/eval_entity_ab.py
python scripts/eval_scorecard.py
python scripts/eval_chat_chester.py
python scripts/eval_chat_overview.py
```

Or from repo root:

```bash
./scripts/entity-ab.sh    # download model, backfill, export corpus, full A/B
```

## Entity A/B report

See [phase3-entity-ab.md](./phase3-entity-ab.md) (generated by `eval_entity_ab.py`).

Enable hybrid in `backend/.env`:

```
ENABLE_NER_ENTITY_EXTRACT=true
```

Then run `./scripts/entity-ab.sh` (or the `source .venv/bin/activate` steps above).
